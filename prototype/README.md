# Noxe — Layout Playground

Protótipo Vite + React + Tailwind v4 para validar 3 opções de layout antes de codar features.

## Como rodar

```bash
cd prototype
npm install
npm run dev
```

Abra http://localhost:5173 e use o toggle do topo para alternar entre os layouts.

## Layouts

- **A · Three-Pane** — Sidebar (notebooks/tags) + lista de notas + editor. Estilo Inkdrop, familiar.
- **B · Sidebar + Tabs** — Activity rail + explorer (file tree) + editor com tabs + painel direito (outline + backlinks). Estilo Obsidian/VSCode.
- **C · Minimal + Command** — Rail slim de ícones + editor centrado + painel meta + command palette protagonista. Inspirado em Tolaria.

Tudo é mock e tema claro. Stack alinhada com Tolaria (React 19, Vite, Tailwind v4, Phosphor).
